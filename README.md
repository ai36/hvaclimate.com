# HVA Climate Control

HVAC service in Portland and Vancouver - [Live demo](https://hvaclimate-com.vercel.app)

## Before running

```sh
npm i
```

## Dev mode

```sh
gulp
```
or

```sh
npx gulp
```

## Build production version

```sh
gulp build
```
or

```sh
npx gulp build
```

Destination folder: **./dist/**

## Addition information

The **./src/img/** folder contains the original images. They are not placed in the git repository.
